# Aider

A Node.js project

## Version

1.0.0

## Installation

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Edit .env with your configuration
```

## Available Scripts

- `npm run test:learn` - cypress run --reporter ./cypress/plugins/windsurf-reporter.js
- `npm run test:analyze` - node scripts/analyze-test-results.cjs

## Testing

```bash
# Run tests
npm test

# Run tests with Cypress
npm run test:learn

# Analyze test results
npm run test:analyze
```

## Development

```bash
# Start development server
npm run dev

# Or start production server
npm start
```
